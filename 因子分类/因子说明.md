# 因子说明

本文档说明前端常用核心因子的含义、分级规则与计算来源。数据由 `ZXW策略技术因子生成.ipynb` 写入 `D:\database\signal_daily`，前端经 `可视化/market_data_service.py` 映射中文名后展示。

---

## MAC总（mac_total）

**是什么：** MACD 相关形态的 **0～4 级综合得分**，把红柱、金叉、底背离、顶背离等合并为一条线；**不是** DIF/DEA 简单相加。

**来源：** `ZXW因子/MACD因子.py` → `build_d_class_factor_bundle`；前端归类 **D类** 核心因子。

**前端用法：** 副图显示 0～4 连续值；回测/组合里常见 `mac_total > 0` 表示「有 MAC 类信号」。**MAC总信号**（`tdx_signal_e`）= `mac_total > 0` 的 0/1 版，见 `ZXW因子/通达信强底信号.py`。


| 值     | 含义  | 触发条件                                                 |
| ----- | --- | ---------------------------------------------------- |
| **4** | 最强  | **底背离 & 红柱** 或 **底背离 & 金叉**（`a_plus`）                |
| **3** | 较强  | **红柱** 且近 7 日内有过 **金叉**，且不满足 4 档（`a_only`）           |
| **2** | 中等  | 当日 **金叉** 或 **红柱**（二者之一）                             |
| **1** | 弱信号 | **底背离** 或 **MAC总顶背离**（`top_divergence_in_mac_total`） |
| **0** | 无信号 | 以上都不满足                                               |


**进一步说明**

- **红柱：** MAC 柱 > 0，且高于 MAC 的 60 日均线。
- **金叉：** DIF 上穿 DEA。
- **底背离：** P弧新低初 / P新低延续背离（价格创新低、DIF 未创新低，含 S笔反K 等结构条件）。
- **顶背离（仅进 1 档）：** P弧新高初 / P新高延续背离，不进 2/3/4 档。
- **同组可单独查看的子因子：** DIF、DEA、MAC、红柱、金叉、底背离、底背离&红柱、MAC总顶背离 等。
- **与通达信原式差异：** 通达信原版 1=最强；Python 版 **4=最强、0=无信号**，且 1 档含顶背离。

## K线底（kline_bottom）

**是什么：** 基于 **多周期低点结构**（历史大底、五年底、两年底、年内底/双底等）与 **相对/绝对到位条件**，合成的 **1～4 级抄底结构分**（**4 最强，1 较弱，0 无信号**）。与 **抄底总分**（`bottom_fishing_score`）数值相同。

**来源：** `ZXW因子/抄底因子.py` → `build_bottom_fishing_factor_bundle`；前端归类 **K线底类** 核心因子。

**前端用法：** 副图看连续分级；**K线底信号**（`tdx_signal_k`）= `K线底 > 0` 的 0/1 版，用于五日内六级、总买入信号等。


| 值     | 含义  | 触发条件                                                                 |
| ----- | --- | -------------------------------------------------------------------- |
| **4** | 最强  | **K线底1** 与 **K线底2** 合成后，对应上一 K 底为 **历史大底** 档（`K底要求` + `K底来要求`） |
| **3** | 较强  | 对应 **五年底** 档                                                           |
| **2** | 中等  | 对应 **两年底** 档                                                           |
| **1** | 较弱  | 对应 **年内底** 或 **年内双底** 档                                               |
| **0** | 无信号 | **K线底1** 与 **K线底2** 均为 0；合成规则：`K线底1×K线底2=0` 取 MAX，否则取 MIN              |


**进一步说明**

- **K线底1（相对）：** 在 **K底要求** 成立时（现价 ≤ 相对低顶），看上一 K 底属于哪档大底结构。
- **K线底2（绝对）：** 在 **K底来要求** 成立时（价格回到上一 K 底附近/年内双底特殊规则），同样按结构分档。
- **合成：** 两路都有值时取 **较小档**（更保守）；仅一路有值时取 **较大档**。
- **底层结构示例：** 历史大底（BARSCOUNT≥750 且 L=历史最低）、五年底、两年底、年内双底（S笔反K + 价同比/K同比 等）、年内底 等；依赖 **S笔反K**、低距L 等 K 线结构量。
- **同组子因子：** K线底1、K线底2、K线底数、K底要求、历史大底、五年底、年内双底 等，可单独展开排查。

## 集中总（concentration_total_score）

**是什么：** 筹码 **集中度** 的综合评分（**0～3**），同时考虑 **相对集中度**（相对自身历史）与 **绝对集中度**（相对全程高低价区间），数值越高表示筹码越 **分散**（集中度数值越大）。

**来源：** `ZXW因子/筹码结构因子.py` → `build_chip_structure_factor_bundle`；需 OHLCV + 换手率矩阵；前端归类 **筹码集中度类** 核心因子。

**前端用法：** 副图看 0～3；**集中总信号**（`tdx_signal_a`）= `集中总 > 0` 的 0/1 版，用于五日内六级、总买入信号改（可加分）等。


| 值     | 含义  | 触发条件                                                                                  |
| ----- | --- | ------------------------------------------------------------------------------------- |
| **3** | 最强  | 相对集中度分、绝对集中度分 **均 > 0**，取 **MIN**；且较小值对应集中度 **(20, 30]**（`_score_by_threshold`） |
| **2** | 中等  | 同上取 MIN，较小值对应集中度 **(10, 20]**                                                        |
| **1** | 较弱  | 同上取 MIN，较小值对应集中度 **≤ 10**                                                            |
| **0** | 无信号 | 相对分、绝对分 **均为 0**；仅一侧 > 0 时取 **MAX**                                               |


**进一步说明**

- **绝对集中度：** `(COST95 - COST5) × 100 / (累计最高 - 累计最低)`，衡量当前筹码在全历史价格区间中的宽窄。
- **相对集中度：** 绝对集中度在约 1200 日（或 BARSCOUNT）窗口内做归一化后的值。
- **分档：** 集中度 ≤10 → 1 分；(10,20] → 2 分；(20,30] → 3 分；>30 → 0 分。
- **集中总合成（通达信对齐）：** 相对分、绝对分 **都 > 0** 时取 **MIN**（取更保守/更低档）；否则取 **MAX**。
- **同组子因子：** 相对集中度、绝对集中度、相对集中度赋值、集中度绝级 等。

## 均线类（ma_class_zxw）

**是什么：** 根据 **多条均线排列、250 日线压制/支撑、支撑计数、初均波动** 等，对当前趋势结构打的 **0～6 分**（**1 最强多头，6 最弱/偏空结构，0 无有效结构或全破**）。

**来源：** `ZXW因子/均线因子.py` → `build_moving_average_factor_bundle`；前端归类 **均线类** 组内 **均线类** 因子。

**前端用法：** 副图看 0～6 连续值。**注意：** 买入信号组里的 **「均线类信号」** 对应 `tdx_signal_i`（`均线类 > 0` 的 0/1 版）；「均线类」组下的 **均线类** 才是本列 `ma_class_zxw` 分级本身。


| 值     | 含义  | 触发条件                                                                                      |
| ----- | --- | ----------------------------------------------------------------------------------------- |
| **1** | 最强  | **多头A加**（C>MA13>MA34>MA60>MA120>MA250）且 **非全破**                                           |
| **2** | 较强  | **多A**（C ≥ 均线最大值）且 **非全破**                                                                |
| **3** | 中等  | **只有250压1** 或 **中多支**（支撑数≥4 且非全空头）且 **非全破**                                              |
| **4** | 偏弱  | **只有250压21** 或 **大2均支撑** 且 **非全破**                                                       |
| **5** | 较弱  | **只有250压23** 或 **小60多** 或 **只250压1支平** 且 **非全破**                                         |
| **6** | 最弱  | **只有250压22** 或 **年半双压31** 或 **小支** 或 **小30多** 且 **非全破**                                 |
| **0** | 无信号 | 以上均不满足，或 **全破**（C < 均线最小值×0.975）                                                         |


**进一步说明**

- **均线集：** MA13/20/21/30/34/60/120/180/250；**初均波动** = MA13/20/30/60 的 20 日标准差相对 MA20 的最大值。
- **非全破：** 收盘价未跌破「均线最小值×0.975」，否则整日为 0。
- **支撑数：** C 站上 MA13/21/34/60/120/250 的条数之和；**全空头** = MA13≤MA34≤MA60≤MA120≤MA250。
- **优先级：** 按 1→6 顺序匹配，命中第一档即输出，不再向下混合。
- **同组子因子：** 多头A加、多A、只有250压1、支撑1～6、非全破 等，可用于拆解当日为何落在该档。

## KDJ超类 / KDJ信号（r_condition）

**是什么：** 标准 KDJ 的 **J 值超卖** 二值信号：J 进入低位区视为「超卖区有信号」。

**来源：** `ZXW因子/KDJ因子.py` → `build_kdj_factor_bundle`；`r_condition` 与 **J值超卖因子** 计算相同（`j_raw < 30`）；前端归类 **KDJ类** 核心 **KDJ信号**。

**前端用法：** 副图 0/1；**KDJ超卖R信号**（`tdx_signal_r`）= `KDJ信号 > 0`，为五日内六级第六路 **R** 条件。


| 值     | 含义  | 触发条件                    |
| ----- | --- | ----------------------- |
| **1** | 有信号 | **J值 < 30**（`j_raw < 30`） |
| **0** | 无信号 | J值 ≥ 30                  |


**进一步说明**

- **J 值计算：** RSV = (C - LLV(L,9)) / (HHV(H,9) - LLV(L,9)) × 100；K、D 为通达信 SMA(3,1)；**J = 3K - 2D**。
- **同组子因子：** RSV、K值、D值、J值（连续数值，非 0/1）。
- **与 J值超卖因子：** 逻辑完全一致，列名不同；组合规则里按英文名 `j_oversold_factor` / 中文「J值超卖因子」引用。

## 逃顶因子 / 逃顶总分（top_escape_score）

**是什么：** **K线底** 的 **镜像顶部分**——用高点结构（历史大顶、五年顶、两年顶、年内顶/双顶等）与 **K线顶1/K线顶2** 合成，输出 **1～4 级逃顶结构分**（**4 最强，0 无信号**）。**逃顶总分 = K线顶** 数值。

**来源：** `ZXW因子/抄底因子.py`（与 K线底 同 bundle）；前端归类 **逃顶类** 核心 **逃顶总分** / **K线顶**。

**前端用法：** 副图看连续分级；卖出组合里常见 **逃顶总分 > 0**（如「卖K线顶」= 当日有正逃顶分），常与 J值超买、MAC总 等 AND 组合。


| 值     | 含义  | 触发条件                                                                 |
| ----- | --- | -------------------------------------------------------------------- |
| **4** | 最强  | **K线顶1** 与 **K线顶2** 合成后，对应上一 K 顶为 **历史大顶** 档（`K顶要求` + `K顶来要求`） |
| **3** | 较强  | 对应 **五年顶** 档                                                           |
| **2** | 中等  | 对应 **两年顶** 档                                                           |
| **1** | 较弱  | 对应 **年内顶** 或 **年内双顶** 档                                               |
| **0** | 无信号 | **K线顶1** 与 **K线顶2** 均为 0；合成规则与 K线底 镜像                                              |


**进一步说明**

- **K线顶1（相对）：** **K顶要求** 下，相对高底 + 上一 K 顶结构分档（对标 K线底1）。
- **K线顶2（绝对）：** **K顶来要求** 下，价格回到上一 K 顶附近的绝对到位（对标 K线底2）。
- **合成规则：** 与 K线底 相同——乘积为 0 取 MAX，否则取 MIN；再映射为 **1 最弱～4 最强**。
- **同组子因子：** K线顶、K线顶1、K线顶2、历史大顶、五年顶、年内双顶 等。

## KDJ超买 / J值超买因子（j_overbought_factor）

**是什么：** J 值进入 **超买区** 的二值卖点辅助信号。

**来源：** `ZXW因子/KDJ因子.py`；前端 **KDJ类** 子因子 **J值超买因子**。

**前端用法：** 副图 0/1；常用于卖出组合，如「MAC总 + 逃顶总分 + J值超买因子」三路 AND，或「总卖出信号 + 逃顶总分 + J值超买因子」。


| 值     | 含义  | 触发条件                    |
| ----- | --- | ----------------------- |
| **1** | 有信号 | **J值 > 70**（`j_raw > 70`） |
| **0** | 无信号 | J值 ≤ 70                  |


**进一步说明**

- 与 **KDJ超卖** 对称，仅阈值方向相反（70 为上轨，30 为下轨）。
- 本身不区分 J 从下方上穿 70 还是持续在 70 以上；当日 J>70 即为 1。

## KDJ超卖 / J值超卖因子（j_oversold_factor）

**是什么：** J 值进入 **超卖区** 的二值买点辅助信号。

**来源：** `ZXW因子/KDJ因子.py`；前端 **KDJ类** 子因子 **J值超卖因子**。

**前端用法：** 副图 0/1；与 **KDJ信号** 数值相同，组合规则中按英文名引用。


| 值     | 含义  | 触发条件                    |
| ----- | --- | ----------------------- |
| **1** | 有信号 | **J值 < 30**（`j_raw < 30`） |
| **0** | 无信号 | J值 ≥ 30                  |


**进一步说明**

- 与 **KDJ超类/KDJ信号** 完全同逻辑；区别主要在命名场景：信号类用「KDJ信号 / KDJ超卖R信号」，规则引擎用「J值超卖因子」。
- ZXW 卖出组合里也可能用 **RSI死叉+RSI超买**，与 KDJ 超卖无关，勿混淆。

## 附录：五日内六级与上述因子的关系

**五日内六级**（`tdx_five_day_level6`）= 近 5 日 rolling max(**超强底**)；**超强底** = 六路同时满足：

| 代号 | 组合信号 | 底层要求 |
| --- | --- | --- |
| E | MAC总信号 | `mac_total > 0` |
| K | K线底信号 | `kline_bottom > 0` |
| A | 集中总信号 | `concentration_total_score > 0` |
| B | 筹码峰信号 | `chip_peak_score > 0` |
| I | 均线类信号 | `ma_class_zxw > 0` |
| R | KDJ超卖R信号 | `r_condition > 0`（J<30） |

另需 K 线根数 ≥ 250（`BARSCOUNT`）。详见 `ZXW因子/通达信强底信号.py`、`backtrader/models/五日内六级_流程图.html`。
